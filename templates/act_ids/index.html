Act ids validation
{% endblock %} {% block content %}
This page lets you validate the ids of an imported act or modify / check the ids of a validated act. Acts can be (re)validated one at a time.
Select an act to validate in the drop down list to validate a new act. Fill in the {{ display_name.releve_annee}}, {{ display_name.releve_mois}}, {{ display_name.no_ordre}} textboxes and click on the modif button to modify an already validated act.
You can then check the data in the table below. Contentious data are highlighted in red. The last row "VALIDATION" contains the final data of the act after validation. Change any value on this row if you detect a mistake. You can also click on the update button should you like to fetch the ids from eurlex and oeil once again.
Under the table, you can add a note concerning the act. Eventually, click on the save button to validate and save the act.
