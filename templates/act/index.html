This page lets you display, modify and save retrieved data of an act (if its ids have been previously validated). Acts data can be displayed one at a time.
Select an act to validate in the drop down list to validate a new act. Fill in the {{ display_name.releve_annee}}, {{ display_name.releve_mois}}, {{ display_name.no_ordre}} textboxes and click on the modif button to modify an already validated act.
You can then check the data in the table below. Change any value (second column) if you detect a mistake. Eventually, click on the save button to validate and save the act data.
